# 案件の追加方法

「案件」とは、kuzen ダッシュボード上の 1 サービス（= 1 クライアント）を指します。  
案件ごとに「どのサービスの CSV を」「どのスプレッドシートのどのシートへ」転写するかを設定します。

---

## 案件の追加手順

1. サイドバーの **「📋 案件管理」** を開く
2. 左上の **「＋ 新規追加」** をクリック → 左リストに「新規案件」が追加される
3. 右側のフォームに各項目を入力する（下表参照）
4. **「保存」** ボタンをクリック

---

## 入力項目の説明

| 項目 | 説明 | 例 |
|---|---|---|
| 案件名 | 画面上の表示名（自由入力） | `TAPP` |
| サービスID | kuzen のサービス ID（4〜8 桁の数字） | `108748` |
| スプレッドシートID | 転写先スプレッドシートの ID | `1EpFtN_82oEQ9Pu...` |
| シート名 | 転写先のシート名（タブ名と完全一致） | `raw_user` |
| コメント | 任意のメモ | 空欄でも可 |

---

## サービス ID の確認方法

kuzen ダッシュボードで対象サービスを開き、URL を確認します。

```
https://dashboard.kuzen.io/services/108748/users?type=userList
                                    ^^^^^^
                                    これがサービス ID
```

---

## スプレッドシート ID の確認方法

Google スプレッドシートの URL から確認します。

```
https://docs.google.com/spreadsheets/d/1EpFtN_82oEQ9PuYqhhXNDSVM0DC4Myz5X2lniaf2Htk/edit
                                       ^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^
                                       これがスプレッドシート ID
```

---

## 有効 / 無効の切替

「全件実行」では **有効** な案件のみ処理されます。  
一時的に除外したい案件は、一覧のトグルスイッチで **無効** に切り替えてください（削除せずに保持できます）。

---

## 案件の編集・削除

- **編集**: 一覧から案件を選択 → 右フォームを修正 → 「保存」
- **削除**: 一覧から案件を選択 → ツールバーの **「🗑 削除」** → 確認ダイアログで「はい」

> 削除は元に戻せません。一時的に不要な場合は「無効」にすることをお勧めします。

---

## 旧バージョン（v1）からの一括インポート

v1 の `.py` 設定ファイルをそのまま読み込めます。

1. ツールバーの **「📥 .py インポート」** をクリック
2. ダイアログが開いたら **「archive_v1 を自動スキャン」** をクリック  
   （または「ファイルを選択…」で個別に選択）
3. インポートする案件にチェックを入れて **「インポート」** をクリック

同じサービス ID がすでに存在する場合は上書き確認が表示されます。
